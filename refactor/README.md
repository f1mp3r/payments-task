# Refactoring Task

## Requirements
- PHP 8.0+
- Composer

## Installation
1. Clone the repository
2. Run `composer install`
3. Add EXCHANGE_API_KEY and BIN_API_KEY in .env file

## Usage
To run the application:
```bash
php bin/app.php input.csv
```

To run tests:
```bash
vendor/bin/phpunit tests
```
